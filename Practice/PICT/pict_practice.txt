#
# для установки pict нужно скачать pict.exe с https://github.com/microsoft/pict и закинуть его в C:\Windows\System32
# pict запускается через PowerShell (перейти в необходимую папку для работы, зажать Shift и нажать ПКМ для вызова контекстного меню → открыть PowerShell)
# для вывода запятой в CSV файле нужно поменять региональные стандарты (Windows 10):
	1. Через поиск Windows ввести "Региональные стандарты" и открыть их
	2. В табе "Форматы" открыть "Дополнительные параметры"
	3. В открывшемся окне в табе "Числа" найти параметр "Разделитель целой и дробной части" и ввести туда знак точки;
	там же найти параметр "Разделитель элементов" и ввести туда знак запятой
	4. Применить изменения
	*Если не хочеться менять эти параметры, то можно использовать LibreOffice Calc вместо Excel, в нём можно выбирать любые разделители
#

- ввод параметров в файле *.txt (<param>: <value_1>, <value_2>, <value_3>, ect.)
	*обозначение негативного параметра (~<value>) (pict дополнительно создаст комбинации с негативными значениями (в одной комбинации будет
	указано только одно негативное значение))
	*задать условие (IF [<param>] = "<value>" THEN [<param>] = "<value>";) (больше условий в manual https://github.com/microsoft/pict/blob/main/doc/pict.md)
- вывод тестовых комбинаций в PowerShell (pict.exe <name_file_with_values>)
- создание файла с тестовыми комбинациями (pict.exe <name_file_with_values> > <name_new_file>)
	*новый файл можно создать с любым расширением, но для удобства лучше всего использовать расширение *.xls (если тестовые комбинации понадобятся для Postman, то
	нужно сохранить полученный файл как *.CSV с разделителем - запятая)
